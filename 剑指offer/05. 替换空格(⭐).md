## 思路
遍历源字符串生成新字符串，遇到空格时替换成 %20


## 解题

```js
/**
 * @param {string} s
 * @return {string}
 */
var replaceSpace = function(s) {
    let rs = ''
    for (let i = 0; i < s.length; i++) {

        rs += (s[i] === ' ' ? '%20' : s[i])
    }

    return rs
};

```
